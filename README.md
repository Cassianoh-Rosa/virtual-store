#Virtual Store
Projeto de uma loja virtual(fícticia) chamada "Kimb Store" com Produtos eletrônicos diversificados. Feita para treinar minha lógica de programação.

# 🚀 Funcionalidades
- Salvamento do carrinho no LocalStorage.
- Filtro por categoria.
- Busca por nome.
- remoção/adição de determinados produtos de sua escolha em seu carrinho.
- responsividade para diferentes dispositivos.

#🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript
